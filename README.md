# WhatsApp Proto

WhatsApp Protocol Buffer definitions compiled from the official web client.

## Installation

```bash
npm install @raphaelvserafim/whatsapp-proto
```

## Available Message Types

This package includes all WhatsApp protobuf message definitions:

- `ADVDeviceIdentity`
- `WebMessageInfo`
- `Message`
- `MessageKey`
- `ContactMessage`
- `LocationMessage`
- `ExtendedTextMessage`
- And many more...
